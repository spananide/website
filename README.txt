# Spanani.dev

<img src="https://spanani.de/assets/images/bg.jpg" alt="Purple space background" width="100%">

**Minimal. Purple. Always evolving.**

Just a clean personal card + a beautiful clock page – built with pure HTML, CSS and a little bit of love.

Live at → https://spanani.de

### Pages
- `/` – Home` – Name, links and that purple vibe  
- `/clock – Clock` – Big glowing time + date in the same style

### Features
- Fully responsive (looks perfect on phone, tablet, desktop)
- Parallax starfield + gradient sky background
- Glassmorphism social icons with hover glow
- Gradient text effect on the name
- No frameworks, no build tools, < 10 KB CSS
- Zero JavaScript on the main page (only a tiny script on /clock)

### Tech Stack
- HTML5
- CSS3 (Flexbox, backdrop-filter, clamp(), gradients)
- Font Awesome 6 (free icons)
- Google Fonts – Space Grotesk

### Local development
Just open `index.html` in your browser – no server needed.  
Or use any static server:
```bash
npx serve
# or
python -m http.server 8000
